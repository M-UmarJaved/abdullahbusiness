### Adding a Grain Bag

1. Click **"Chaki"** button
2. Find the form **"Add New Bag"**
3. Fill in these fields:

```
Customer Name:      [Write customer name]
```

7. Check **"Recent Bags"** table below to verify it saved

## ğŸš¨ TROUBLESHOOTING - WHAT TO DO IF...

### "The app isn't opening"
1. Check internet connection
2. Refresh browser (press F5)
3. Try different browser (Chrome, Safari, Edge)
4. Clear browser cache (Ctrl+Shift+Delete)

### "My data isn't saving"
1. Check green/red dot (online/offline status)
2. Try clicking **"Add"** again
3. Refresh page
4. Try on computer instead of phone
5. Contact blood relative for help

### "I see a red error message"
1. This is usually temporary
2. Wait 5 seconds
3. Try again
4. If persists, refresh page
5. Contact blood relative

### "The form looks broken on my phone"
1. Rotate phone to landscape
2. Try different phone brand
3. Use computer instead
4. Contact blood relative

### "I accidentally added wrong data"
1. This version doesn't have delete yet
2. Add a correction entry with opposite amount
3. Or ask blood relative to delete from Firebase console

### "I forgot what a field means"
1. Read this guide again
2. Ask blood relative to explain
3. The app has hover tooltips (hover over "?" symbols)
